# Denver — Quick training (5 minutes)

## What this website does

- **Official ADe Garage home** — one link for Facebook, TikTok, and Shopee bios
- Customers **browse parts here**, then **order on Shopee, TikTok Shop, or Messenger**
- **Lazada** shows "coming soon" until your store is live
- **No monthly fee** — you keep using Shopee/TikTok seller admin like today

## What you still manage (same as before)

| Platform | You update products in... |
|---|---|
| Shopee | Shopee Seller Centre |
| TikTok Shop | TikTok Shop Seller |
| Lazada (later) | Lazada Seller Centre |
| Website catalog | Ask Ryan to update, or send new product links |

## When you add a new product

1. List it on Shopee (and TikTok Shop if you want)
2. Send Ryan the **product title + Shopee link + TikTok link** (if any)
3. He adds it to the website catalog

## Fitment questions

Customers can click **"Ask on Messenger"** — same as Facebook DMs today.

## After launch — add this link everywhere

- Facebook page → Website field
- Shopee shop description
- TikTok bio

**Suggested text:**

> Official catalog: [your-domain] — order on Shopee, TikTok Shop, or message us!

## Phase 2 (optional later)

When you're ready for **checkout on your own website** (~$5/month), Ryan adds an "Order on website" button — same site, no rebuild.
